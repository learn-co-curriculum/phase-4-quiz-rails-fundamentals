# Routing: Route Params

Given the following route definition:&nbsp;

`get "/dogs/:id", to: "dogs#show"`

When a `GET` request is made to `/dogs/2,` how would you access the ID of 2 from
the URL?

## Correct

params[:id]

## Incorrect

params[2]

## Incorrect

params[id]

## Incorrect

params.id
